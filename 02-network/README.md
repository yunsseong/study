## 최종 목표: "브라우저에 URL을 입력하면 일어나는 일"을 완벽하게 설명하기

```
URL 입력 → DNS 조회 → TCP 3-way handshake → TLS handshake (HTTPS)
→ HTTP 요청 → 서버 처리 → HTTP 응답 → 브라우저 렌더링
```

이 흐름을 모든 계층에서 설명할 수 있으면 네트워크 면접 준비 완료.
