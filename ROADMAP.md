## Phase 1: 핵심 기초

### 1. [자료구조 & 알고리즘](./01-data-structures-algorithms/)

코딩테스트 통과가 첫 관문.

| 순서 | 주제 | 핵심 포인트 |
|------|------|-------------|
| 1 | 배열, 문자열 | 투 포인터, 슬라이딩 윈도우 |
| 2 | 스택, 큐 | 괄호 문제, BFS |
| 3 | 해시맵 | 빈도수 카운팅, 중복 검사 |
| 4 | 트리, 그래프 | DFS/BFS, 이진탐색트리 |
| 5 | 정렬, 이진탐색 | 시간복잡도 비교 |
| 6 | DP, 그리디 | 점화식 세우기 |
| 7 | 시간/공간 복잡도 | Big-O 분석 |

**연습**: 백준, 프로그래머스 하루 1-2문제

---

### 2. [네트워크](./02-network/)

백엔드 개발자에게 특히 중요한 분야.

- **TCP vs UDP** - 3-way handshake, 흐름/혼잡 제어
- **HTTP/HTTPS** - 메서드, 상태코드, TLS 동작 원리
- **REST API** - 설계 원칙, 멱등성
- **DNS 동작 과정** - 브라우저에 URL 입력하면 일어나는 일
- **쿠키 vs 세션 vs 토큰(JWT)**
- **웹소켓** - 실시간 통신

---

### 3. [운영체제](./03-operating-system/)

면접 단골 질문이 가장 많은 분야.

- **프로세스 vs 스레드** - 차이점, 컨텍스트 스위칭
- **동기화** - 뮤텍스, 세마포어, 데드락 조건 4가지
- **메모리 관리** - 가상 메모리, 페이징, 세그멘테이션
- **스케줄링** - RR, SJF, Priority 등 알고리즘 비교
- **인터럽트, 시스템 콜** - 유저모드 vs 커널모드

---

### 4. [데이터베이스](./04-database/)

백엔드의 핵심.

- **SQL** - JOIN, 서브쿼리, GROUP BY, 인덱스 활용
- **인덱스** - B-Tree 구조, 왜 빠른지, 어떤 컬럼에 걸어야 하는지
- **트랜잭션** - ACID, 격리 수준 4단계
- **정규화** - 1NF~3NF, 반정규화를 하는 이유
- **NoSQL vs RDB** - 언제 어떤 걸 쓰는지
- **쿼리 최적화** - 실행계획, 슬로우 쿼리 분석

---

## Phase 2: 백엔드 심화

### 5. [시스템 설계](./05-system-design/)

경력/신입 가리지 않고 점점 중요해지는 분야.

- **캐싱** - Redis, 캐시 전략 (Cache Aside, Write Through)
- **로드밸런싱** - L4 vs L7, 알고리즘
- **메시지 큐** - Kafka, RabbitMQ 개념
- **DB 확장** - 레플리케이션, 샤딩, 파티셔닝
- **API 설계** - 인증/인가, Rate Limiting, 페이지네이션

---

### 6. [소프트웨어 공학 & 설계](./06-software-engineering/)

좋은 코드를 작성하기 위한 기반.

- **디자인 패턴** - 싱글톤, 팩토리, 옵저버, 전략 패턴
- **SOLID 원칙** - 각 원칙의 실제 적용 예시
- **클린 아키텍처** - 레이어드 아키텍처, 의존성 역전

---

### 7. [보안](./07-security/)

백엔드 개발자 필수 지식.

- **인증/인가** - OAuth 2.0, JWT 동작 원리
- **주요 취약점** - SQL Injection, XSS, CSRF
- **암호화** - 대칭키 vs 비대칭키, 해싱 (bcrypt)

## Phase 3: 프레임워크 실습

### 8. [Java Spring Backend](./08-java-spring/)

대기업, 금융, 엔터프라이즈 시장의 주력 기술 스택.

- **Java 기초 + OOP** → Spring Core (IoC, DI, AOP)
- **Spring Boot + MVC** → REST API 설계
- **Spring Data JPA** → 영속성 컨텍스트, N+1 문제
- **Spring Security** → JWT, OAuth2
- **테스트 + 배포** → JUnit, Docker

### 9. [FastAPI Backend](./09-fastapi/)

스타트업, AI/ML, 빠른 프로토타이핑에 강한 기술 스택.

- **Python 핵심 + 타입힌트** → FastAPI Core
- **Pydantic** → 데이터 검증, 직렬화
- **SQLAlchemy + Alembic** → ORM, 마이그레이션
- **JWT 인증 + 비동기** → async/await 심화
- **테스트 + 배포** → pytest, Docker
