## 백엔드 개발자에게 DB란?

```
백엔드의 대부분의 버그와 성능 문제는 DB에서 발생한다.

코드가 느린 게 아니라 쿼리가 느린 것이고,
데이터가 꼬이는 건 트랜잭션을 모르기 때문이고,
서비스가 커지면 인덱스와 정규화를 다시 보게 된다.
```
