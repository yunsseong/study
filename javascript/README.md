# 자바스크립트 마스터 커리큘럼

8-10주 집중 부트캠프 스타일 학습 과정

## 📅 커리큘럼 구조

### Week 1-2: 자바스크립트 코어 기초
- 변수 (var/let/const), 호이스팅
- 데이터 타입, 타입 변환
- 함수 (선언식, 표현식, 화살표 함수)
- 스코프와 클로저
- this 키워드

### Week 3: 비동기 프로그래밍
- 이벤트 루프와 콜스택
- 콜백, Promise, async/await
- 에러 핸들링

### Week 4: 브라우저 & DOM
- DOM API
- 이벤트 처리
- 브라우저 저장소
- Fetch API

### Week 5: 모던 자바스크립트 (ES6+)
- 구조 분해, 스프레드/레스트
- 템플릿 리터럴
- 모듈 시스템
- 클래스와 상속

### Week 6-7: Node.js & 백엔드
- Node.js 런타임
- Express.js
- RESTful API
- 인증/보안

### Week 8: React 기초
- 컴포넌트, Props, State
- Hooks (useState, useEffect)
- 빌드 도구

### Week 9-10: 풀스택 통합 프로젝트
- 프론트-백엔드 연동
- 실전 프로젝트

## 📝 학습 방법

1. 각 주차 폴더의 `lesson` 파일로 이론 학습
2. `exercise` 파일에서 실습 과제 수행
3. `exercise-answer.md`에 답안 작성
4. `quiz` 파일로 이해도 체크
5. `project` 파일로 실전 적용
